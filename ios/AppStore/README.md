# iOS App Store Submission Materials

This directory contains all materials needed for iOS App Store submission.

## Contents

### App Icons
- `AppIcon/` - Contains all required app icon sizes for iOS devices
- `AppIcon/Icon-1024.png` - App Store marketing icon (1024x1024)

### Screenshots
- `Screenshots/` - Device-specific screenshots for App Store listing
- `Screenshots/iPhone/` - iPhone screenshots (6.7", 6.5", 5.5")
- `Screenshots/iPad/` - iPad screenshots (12.9", 11")

### Metadata
- `AppStoreMetadata.md` - App Store listing information
- `PrivacyPolicy.md` - Privacy policy for App Store compliance
- `AppStoreGuidelines.md` - Compliance checklist

### TestFlight
- `TestFlightSetup.md` - Beta testing configuration guide
- `BetaTestingNotes.md` - Release notes for beta testers
